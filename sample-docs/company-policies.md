# Acme Inc. — Company Policies & Culture

## About Acme Inc.
Acme Inc. is a B2B SaaS company building project management tools for remote engineering teams. Founded in 2021, we're a team of 65 people across 12 countries. Our product serves over 2,000 companies worldwide.

## Mission
Make remote engineering teams 10x more productive by eliminating coordination overhead.

## Core Values
1. **Ship fast, iterate faster** — We prefer working software over perfect plans
2. **Default to transparency** — Share context broadly, make decisions visible
3. **Own the outcome** — Don't just complete tasks, solve problems
4. **Remote-first, not remote-only** — We optimize for async work but value human connection

## Work Schedule & Time Off

### Working Hours
- We're a remote-first company. Work when you're most productive.
- **Core overlap hours**: 10:00 AM – 2:00 PM EST (for meetings and real-time collaboration)
- Outside core hours, default to async communication (Slack messages, Linear comments, Loom videos)

### PTO Policy
- **Unlimited PTO** with a minimum of 15 days/year
- Please give at least 2 weeks notice for time off > 3 days
- Log time off in BambooHR and post in `#out-of-office` Slack channel
- No approval needed for < 3 days, just notify your manager

### Holidays
- We observe 10 US federal holidays
- Employees outside the US can swap US holidays for local holidays

## Security Policies

### Access Management
- All company accounts require SSO via Okta
- Enable 2FA/MFA on all work accounts (enforced via Okta policy)
- Use 1Password Teams for password management (company-provided)
- Access is role-based (RBAC). Request additional access through #infra-requests

### Data Handling
- Never store customer data on personal devices
- Use encrypted channels (Slack, email) for sensitive information
- All code must be in the company GitHub org — no personal repos for work code
- Report security concerns to security@acme-inc.com immediately

### Device Security
- Enable FileVault (macOS) or BitLocker (Windows) disk encryption
- Keep OS and software up to date (auto-updates enabled via MDM)
- Lock your screen when stepping away (even at home)
- Don't install unvetted software — check with #it-support first
